<h4><b>How To Run</b></h4>
<p>
To run particular episode Change line number 11 in Program.cs file to the desried episode class name
</p>

<div>
For example to run episode 2, the code should be
<code>

    using (var game = new Episode2())
        game.Run();

</code>
</div>
